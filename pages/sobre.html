Informações sobre o projeto.

Este projeto é uma Single Page Application (SPA) desenvolvida em JavaScript.

A SPA permite a navegação entre diferentes seções sem recarregar a página inteira, proporcionando uma experiência de usuário mais fluida e rápida.

Utilizamos técnicas de manipulação do DOM e roteamento baseado em hash para gerenciar o conteúdo exibido.

Explore as diferentes seções usando o menu de navegação acima para aprender mais sobre como funciona uma SPA.

Esperamos que você aproveite a experiência!
